Hosting Your Website

A simple guide to hosting your website on a server.

Hosting

Hosting refers to the process of storing, serving, and maintaining files for a website or web application on a server. When you create a website, you need a place to store its files, including HTML, CSS, JavaScript, images, and other assets. This is where hosting comes into play. A hosting service provides the infrastructure and resources needed to make your website accessible on the internet.

Configuring IIS

  1. Click on Start → Control Panel → In the Programs category, Click on Get Programs → Click on Turn Windows Features on or off → A new dialog box will appear in which select the Internet Information Services → Click Ok
  2. It will install Internet Information Services on your computer, and a folder named INETPUB will be created on C drive
  3. Now check IIS by opening Internet Explorer and typing the address http://localhost or http://ipaddress (IP address of your system) or http://computername (Your computer name) or http://fullcomputernamewithdomain (e.g., http://cl-101.jiet.co.in)
  4. A default page named IISstart.html will open.
  5. Create a web page named index.html and copy it inside the directory C:\Inetpub\wwwroot
  6. Now check your home page by opening the local site as done in step 3. Now your home page will open
  7. By default, the default physical path for the website is C:\Inetpub\wwwroot. You can add your folder as a default physical path.
  8. Create a folder named as yourname inside the folder C:\Inetpub\wwwroot\ and cut index.html file from the folder wwwroot and paste it inside the folder yourname (like ram, mohan, etc.). Now the physical path of your website is now C:\Inetpub\wwwroot\yourname
  9. In order to change the default website path to your desired website path, Click on Start → Control Panel → Click on drop-down icon on Viewed by Category → Click on Small Icons → Click on Administrative Tools → click on Internet Information Services Manager → Click on computer name in the left side in the Connections menu → Click on sites → Select Basic setting in the options given at the right side.
  10. A new dialog box named as Edit site will open. Change the physical path from %SystemDrive%\inetpub\wwwroot to %SystemDrive%\inetpub\wwwroot\yourname → Click ok
  11. Now again check your home page by opening the local site as done in step 3. Now your home page will open
